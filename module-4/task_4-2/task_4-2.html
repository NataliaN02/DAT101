1. What is an array?

    2. An array is a list-like structure in programming that can store multiple values in a single variable, accessible by an index.

  2. What does it mean to "hardcode" when we talk about programming?

    3. Hardcoding means writing values directly into the code instead of generating them dynamically or getting them from input or configuration.

  3. Explain in plain text what happens in task 3, and how we can traverse (iterate) the array when we (humans) don't "know" how long the array is.

    4. In task 3, we take a string, split it into words, and store them in an array. We traverse the array using a loop (like for or forEach) to print each word and its index. This works even if we don’t know the length beforehand because arrays have a .length property that tells us how many elements exist

  4. Why is it beneficial to store instances of a class in arrays, as in task 4?

    5. Storing class instances in arrays is beneficial because it allows easy management of multiple objects. We can loop through them, remove elements, or perform operations on all instances without writing repetitive code.

  6. In task 8, we will use a "callback" routine when sorting. Explain how such a "callback" routine works.

    7. A callback in .sort(...) is a function passed into another function to determine sorting order. It receives two elements and returns a number: negative if the first should come before the second, positive if after, and zero if equal.

  7. What happens in the sorting routines you have created? Try to explain how the array can figure out if an element should come before or after another. The best way to answer this is to actually print both parameters each time the sorting routine runs.

    8. In sorting, the callback compares two elements at a time. The array uses the returned value to decide which element comes first. By printing the parameters in the callback, you can see each comparison and the decision made.

  8. In task 10, we will create an array with rows and columns. Explain what the first element of the array contains.

    9. The first element of the array contains the first row, which itself is an array containing all the columns of that row. For example, array[0][0] is the first cell in the first row, array[0][1] is the second cell in the first row, etc.
