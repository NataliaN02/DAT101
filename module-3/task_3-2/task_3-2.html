  2. 2. In part 2, can we create what is known as an "infinite" loop within the condition of the "while" loop? Describe what would or would not be necessary to achieve this. Provide a thorough explanation of your answer.

    3. A "while" loop can run indefinitely if the condition never becomes false, which is called an infinite loop. In the guessing number program, Math.random() generates a new random number in each loop iteration. If the guessed number is never correct, the while loop keeps running, drawing new numbers again and again. Because Math.random() is pseudorandom, there is no guarantee the correct number will be picked quickly, so the loop could run for a long time. During this, the program is busy with the loop and usually cannot do other tasks at the same time. Nothing extra is needed to make the loop infinite. In order to avoid creating an infinite loop, one usually add for example a maximum number of guesses, to prevent the program from running forever.
